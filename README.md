# Dotfiles

This repository contains all my **dotfiles**, and a script for a fast development environment setup.

## How to use

1. Install [Yadm](https://yadm.io)
2. Clone the repo `yadm clone https://github.com/dapeduu/dotfiles`
3. It will prompt you to use the bootstrap script, so type `y`
4. Make **zsh** the default shell `sudo chsh -s $(which zsh)`
5. Download a [Nerd Font](https://www.nerdfonts.com) to be able to use icons
6. And we're good to go :)

Or

Just run: `curl -Sl https://github.com/dapeduu/dotfiles/blob/master/.config/yadm/bootstrap-helper | bash`
